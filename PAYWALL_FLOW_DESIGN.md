# PAYWALL_FLOW_DESIGN — NoMoreBets 商业化改版总设计

> 状态：设计已定稿（2026-07-04 与开发者逐条确认），待实现。
> 实现档位建议：前端漏斗 + 计费改动 = Opus 4.8 · ULTRACODE（先进 Plan 模式）；90 条每日任务等长文案 = Fable 5。
> 本文档自包含：不看聊天记录也能照此实现。

---

## 0. 核心原则（为什么这么设计）

1. **冷热共情差（hot-cold empathy gap）**：用户付费意愿的顶点 = 深夜输完钱、下定决心、下载打开 App 的那 5 分钟。整个转化流程必须在这一口气里走完，**不给思考间隙**（流程不断 + 出口不可见）。参考 QUITTR：12 题问卷 99% 完成率、下载→付费 25%、4 个月 $250K MRR。
2. **卖的不是工具，是"90 天后的你"**：付费墙展示的是个性化计划和结果承诺，不是功能清单。
3. **真稀缺，不做假稀缺**：划线价、一次性优惠、倒计时都可以有，但必须真实执行（见 §4）。假倒计时 = Apple 3.1.2 风险 + 康复类 App 的口碑雷。
4. **安全底线（不可讨论）**：热线、5 分钟冷静、冲动急救永远在付费墙外。生日/早晚问候等关怀通知免费。
5. **免费 = 安全网，不是货架**：购买时刻用户看不到免费选项；跳过后才掉进薄免费模式（见 §5）。

## 1. 定价梯子（含新增买断）

| 档 | 价格 | 定位 |
|---|---|---|
| 个人自救版 · 月付 | $9.99（可提至 $11.99） | 故意不划算，逼年付 |
| **家庭守护版 · 年付（主推 hero）** | **年付 $59.99；划线锚点 = 按月付一年的真实价格（$9.99×12 = $119.88）→ 显示"省 50%"**。⚠️ 注意：Apple 每个订阅只能有一个 introductory offer，7 天免费试用已占用该名额，所以**不能**再叠加"首年折扣 intro offer"——锚点只能用按月换算价（真实、合规、业内通用做法） | 付费墙默认选中；文案锚定用户自己填的月损失："不到你两天的损失" |
| 互相守护版 · 年付 | $79.99（同样可做 intro offer） | 两个人的溢价档 |
| **终身买断（新增 SKU，非消耗型）** | **$99.99–129.99** | ①价格锚点让年费显便宜 ②收编"死活不订阅"的华人用户 ③戒赌是"好了就走"场景，买断 LTV 未必低于订阅 |
| AI 加购包 | 现有 $9.99 外，增设 **$4.99 小包** | 降低冲动购买门槛 |

试用：全档 7 天免费试用（已在 ASC 配置，App 文案已统一为 7 天）。

4. 平均每个月，大概输多少？ $500以下 / $500–2,000 / $2,000–5,000 / $5,000–10,000 / $10,000以上 → **喂省钱计数器基线（首页不再重复问）**
5. 有没有为了赌借过钱？ 从来没有 / 有过一两次 / 经常 / 现在还背着赌债

8. 上次赢钱之后，你是收手了，还是继续赌到输光？（测追损 chasing）

**Lie/Bet 量表（临床验证，两题即可筛出赌瘾）：**
9. 你有没有对家人隐瞒过你到底赌了多少？
10. 你是不是要赌越来越大的金额才够刺激？

**情绪/身份题（解除防御 + 锁定决心 + 喂承诺书和 AI 人设）：**
11. 赌博已经影响到了哪些？（多选） 存款见底 / 背上债务 / 夫妻关系 / 孩子 / 工作 / 睡眠 / 看不起自己
12. 赌博让你失去过什么、到现在都后悔？（自由填写）
13. 你有过"这次真的不能再这样下去了"的时刻吗——是不是那个时刻让你下载了这个 App？ 是，就是现在 / 有过好几次 / 说不清
14. 如果戒成了，你最想把钱花在谁身上？最想听到谁对你说什么？（自由填写 → 喂承诺书 / 里程碑 / 生日祝福 / AI 人设）

## 3. 付费墙文案（结果导向，替换现有功能清单式）

- 标题：**开启你的 90 天新生计划**
- 副题：头 7 天免费。你每月输 ${用户 Q4 答案}——这个计划一年只要 $59.99，**不到你两天的损失**。
- 三张卡（顺序：家庭守护默认选中 / 互相守护 / 个人自救），每张一句结果语：
  - 家庭守护版：让最爱的人陪你走这 90 天
  - 互相守护版：和另一个人互相拉住对方
  - 个人自救版：一个人也能走完
- 终身买断卡：**「不想订阅？一次付清，永远拥有」$99.99**（放三卡下方，当锚点）
- 主按钮：**免费开始 7 天 · 之后 $59.99/年**
- 跳过：小灰字、延迟 2 秒出现：「暂不开启计划，先随便看看」→ 进薄免费模式

## 4. 真稀缺规则（一次性优惠页）

1. 划线价真实：锚点 = 按月付 12 个月的真实总价（$9.99×12=$119.88 划掉 → 年付 $59.99，省 50%）。不虚构"原价"；7 天免费试用保留（它就是该订阅唯一的 introductory offer，不能再叠加首年折扣）。
2. **一次性出现**：付费墙原生带「这个价格只出现这一次，关掉就恢复原价」，代码真实执行（跳过后再进付费墙 = 显示无折扣原价，或仅在 §5 热时刻重新给一次）。
3. 倒计时（可选）：如做，必须真实——倒数结束本次优惠真失效。**禁止**刷新重置的假倒计时。
4. 不做假"限量"、假"仅剩 X 名额"。

## 5. 薄免费模式（跳过后的安全网）

**永远免费：**
- 冲动急救整页（倒计时/热线/联系人拨打）——合规底线
- 每日打卡 + 连续天数 + 里程碑
- **省钱计数器**（损失厌恶钩子：让"已省 $6,000"天天涨，他舍不得删）
- 每日记录基础表单 + 日历
- 故事墙浏览与投稿（社区要人气）
- 早晚问候 / 生日祝福通知
- 保护卡 1 张/月
- 联系人/目标 各 1 个
- AI 冲动倾诉 **每月 3 次尝鲜**

**Pro 才有：**
- 90 天计划每日任务（付费主菜）
- AI 50/100 次 + 加购
- AI 周分析 / 月报告 / 模式洞察
- 守护邀请（家庭/互相）
- 保护卡 3 张/月
- 联系人/目标无限 + 照片

**再转化炮位（薄免费用户的三个热时刻，温和再弹一次付费墙）：**
1. 记录完「我刚从赌场回来」复赌流程后：「别让这次白疼。开启你的 90 天计划」（第二热的时刻）
2. 发薪日提醒点进 App 时
3. 连续天数到 7 天：「用计划保住这 7 天」

## 6. 赌徒心理 → 产品武器对照

| 机制 | 用法 |
|---|---|
| 追损 chasing | 结果页 5 年投影量化那个洞；Q8 测出追损型给专属文案 |
| 损失厌恶（痛≈2×爽） | 省钱计数器免费且醒目；保护卡防清零焦虑（Pro 3 张） |
| 沉没成本 | 14 题问卷=投入感；答完不舍得走 |
| 羞耻与隐瞒 | 访客模式保匿名；全程不评判语气；Q9 隐瞒题解除防御 |
| 触底时刻 | 漏斗卡首开 5 分钟；复赌记录后是第二炮位 |
| 身份与希望 | 卖"90 天后的你"；承诺书用用户自己的话 |

## 8. 合规红线

- 急救功能永远免费可达（Apple 健康类 + 良心）。
- 不做假倒计时/假限量；折扣走 Apple introductory offer。
- 诊断措辞避免医疗断言（"和你相似的人的经验"≠"医学诊断"）。
- 订阅条款完整披露（试用后自动续订、Apple ID 扣款、删 App 不退订）——现有 PaywallModal 底部披露保留。
- 问卷含自伤类答案时走危机分流（988/911，复用 containsSelfHarm）。
